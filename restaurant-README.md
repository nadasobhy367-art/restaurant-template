## ✨ Features

- 🎨 Elegant design with warm restaurant colors
- 📱 Fully responsive on all devices
- 🍕 Interactive menu with category tabs
- 🖼️ Photo gallery with lightbox
- 📅 Reservation form section
- 🌍 Arabic RTL support
- ⚡ Smooth scroll animations
